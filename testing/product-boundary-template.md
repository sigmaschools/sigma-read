## Quality Bar
_Every improvement must pass these filters:_

1. **Scope check:** Does this strengthen the core loop (read → discuss → assess)?
2. **User check:** Does this serve guides or students directly?
3. **Impact check:** Would >50% of users encounter this?
4. **Simplicity check:** Does this make the app simpler, not more complex?

If any answer is "no," the improvement needs explicit founder approval.
